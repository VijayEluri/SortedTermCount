Welcome!
Given an order-sorted signature S and a closed formula F, this application will produce a bound, based on F's Skolem signature, for each sort. Every sort's bound, if finite, is sufficient (with respect to that sort) for checking F's satisfiability via model finding. In fact, all sentences with the same Skolem signature will produce the same bounds. For more detail on the setting, the theory, and algorithms involved, see our technical report.

Enter one constant or function per line as name followed by signature.
For instance, "f A B C" means that f is a function from (A x B) to C, where A, B, and C are sorts.
